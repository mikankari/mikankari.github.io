制作物

  • 個人的タスク管理

    HTML5Electron個人制作

    チケット駆動開発のためのマルチタスク管理・自動化ツール。着手するタスクをすばやく決められる。日報を一瞬で作成・投稿する。

  • Twitter for Brackets

    HTML5個人制作

    Adobe Brackets の拡張機能。他のエディタにあるのに Brackets にはないので作った。Twitter のタイムライン閲覧とツイート投稿ができる。

  • 紙メディアから Web サイトを制作するアプリケーション

    C++OpenCV研究

    知識なく安価で自由なデザインの Web サイトを制作できる。紙に書いた図形と貼った付箋をスキャンして、HTML・CSS を生成する。

  • Live Preview for Responsive　

    HTML5研究

    Adobe Brackets の拡張機能。２つのインラインフレームによる複数プレビューと、コードの変更をリアルタイムに反映する。

  • Mascot-Share

    HTML5PHPMySQL課題制作

    デスクトップマスコットキャラクターを投稿して、他のユーザに共有し、使ってもらう。他のユーザに評価をしてもらうこともできる。

  • Rhyth/Walk

    Androidチーム製作

    ミライケータイプロジェクトにて、はこだて未来大学との共同開発。ユーザの現在のシチュエーションにマッチした音楽を選曲、再生する。

  • ヤンデレんこ

    HTML5Socket.IO課題制作

    AIを持ったヤンデレなキャラクターが、他のマシンのAIと戦う。防御態勢をとると他のデバイスに逃げる。
